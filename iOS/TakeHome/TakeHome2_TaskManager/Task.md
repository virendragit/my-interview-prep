American Airlines Coding Exercise
Please begin by creating a new iOS application project in Xcode. The purpose of this
application is to allow the user to manage a list of tasks. The application should have the
following functionality:
• View a list of tasks
• Add a new task
• Toggle task completion
• Delete a task

### Lets Split into smaller Tasks:
1. Create MVVM structure of the poject.
2. Creat VM
3. Define functionality
4. Creating UI in Swift UI , if required will try to add concurancy for smooth UI.
5. Unit Testing.
6. UI Testing.
7. If time allows documenting.
8. Adhare with SOLID Design Principle.
9. Check where we can utilize the oops principles.
